# UI/UX Pro Max 快速开始

## 🚀 立即开始重构前端

### 方式一：直接描述需求（最简单）

直接在 Antigravity 中输入：

```
重构前端界面，使用现代化的专业设计风格
```

Agent 会自动：
- ✅ 搜索适合的设计系统
- ✅ 应用配色方案
- ✅ 优化组件样式
- ✅ 添加动画效果

---

### 方式二：指定具体页面/组件

```
重构聊天页面（web/pages/chat/index.tsx），应用 Glassmorphism 风格，使用 SaaS 配色方案
```

```
优化首页的卡片组件，提升视觉层次和设计感
```

```
改造侧边栏导航，使用现代化的设计，添加平滑过渡动画
```

---

### 方式三：完整设计系统升级

```
对整个前端项目进行现代化改造：
- 应用 Glassmorphism + Minimalism 风格
- 统一使用 SaaS 数据产品的配色方案
- 使用 Poppins + Open Sans 字体配对
- 优化所有组件的视觉层次
- 添加专业的动画和交互效果
- 确保响应式和可访问性
```

---

## 📋 常用提示词模板

### 重构单个页面
```
重构 [页面路径]，要求：
- 风格：现代化、专业、有设计感
- 配色：SaaS 数据产品风格
- 动画：平滑过渡、微交互
- 响应式：移动端友好
```

### 优化组件
```
优化 [组件名称]，应用现代化的设计风格，提升用户体验和视觉层次
```

### 应用设计系统
```
将 UI/UX Pro Max 的设计系统应用到 [页面/组件]，确保视觉一致性和专业的设计感
```

## 🎨 可用的设计风格关键词

- **Glassmorphism** - 毛玻璃效果
- **Minimalism** - 极简主义
- **Modern Professional** - 现代专业
- **SaaS** - SaaS 产品风格
- **Data Analytics** - 数据仪表板风格
- **Tech Startup** - 科技创业公司风格

## 💡 实际使用示例

### 示例 1：重构聊天界面
```
使用 UI/UX Pro Max 重构聊天界面：
1. 搜索适合聊天界面的设计风格
2. 应用 Glassmorphism 到消息容器
3. 优化消息气泡设计
4. 添加发送动画
5. 改进输入框样式
6. 确保暗色模式体验
```

### 示例 2：优化首页
```
优化首页（web/pages/index.tsx）：
- 应用现代化的卡片设计
- 改进搜索框和筛选器样式
- 添加平滑的加载动画
- 优化响应式布局
```

### 示例 3：改造导航
```
改造侧边栏导航（web/components/layout/side-bar.tsx）：
- 使用现代化的导航样式
- 添加激活状态的视觉反馈
- 优化图标和文字间距
- 添加平滑的过渡动画
```
